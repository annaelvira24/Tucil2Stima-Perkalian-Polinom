Pada folder src, terdapat 4 buah source code utama, yaitu:
- array.h
- PolinomBruteForce.h
- PolinomDivideConquer.h
- main.cpp

Pustaka terdapat pada file PolinomBruteForce.h dan PolinomDivideConquer.h
sementara file main.cpp merupakan program utama yang akan memuat (include)
PolinomBruteforce.h dan PolinomDivideConquer.h untuk menguji program 
perkalian polinom dalam kedua pustaka tersebut

-------------------------------------------------------------------------------
Cara kompilasi di Windows dan Linux
g++ array.h PolinomBruteForce.h PolinomDivideConquer.h main.cpp -o main

-------------------------------------------------------------------------------
Pada folder bin, terdapat executable file untuk windows dengan nama main.exe
FExecuteable file tersebut dapat langsung dijalankan
